# Inteli Spaces in VR

![Current project](https://i.giphy.com/media/v1.Y2lkPTc5MGI3NjExcjg3bGRscGR0enZ2ZG0wMWxyaXhld2tuNXFzM2o0bjFvMzg0dW8yZyZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/vdjJ0dsgVVsXiypRzE/giphy-downsized-large.gif)

This project creates a simple virtual reality scene using [A-Frame](https://aframe.io/), displaying 360-degree views of the Inteli environments. The scene features a skybox with a panoramic image and a text label.

## Project Structure

The project consists of a single HTML file that contains the A-Frame scene. It includes:
- A sky (`<a-sky>`) element that uses a 360-degree panoramic image of the Inteli Lab.
- A text (`<a-text>`) element that displays the label "Inteli Lab" within the scene.

## Getting Started

### Prerequisites

- A web browser that supports WebVR, such as [Meta Quest Browser](https://www.meta.com/quest/).
- An HTTP server to serve the HTML file for local or online testing.
